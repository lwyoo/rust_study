# 복합 할당 연산자 (Compound Assignment Operators)

기존 값에 연산을 적용하면서 동시에 할당하는 연산자입니다.

## 📋 종류

| 연산자 | 의미 | 예제 | 동일 표현 |
|--------|------|------|----------|
| `+=` | 더하고 할당 | `x += 5;` | `x = x + 5;` |
| `-=` | 빼고 할당 | `x -= 3;` | `x = x - 3;` |
| `*=` | 곱하고 할당 | `x *= 2;` | `x = x * 2;` |
| `/=` | 나누고 할당 | `x /= 4;` | `x = x / 4;` |
| `%=` | 나머지 할당 | `x %= 3;` | `x = x % 3;` |

---

## 💻 실제 코드 예제

```rust
fn main() {
    // ➕ += 연산자
    let mut x = 10;
    x += 5;
    println!("x += 5: {}", x);  // 출력: x += 5: 15
    
    // ➖ -= 연산자
    let mut y = 20;
    y -= 3;
    println!("y -= 3: {}", y);  // 출력: y -= 3: 17
    
    // ✖️ *= 연산자
    let mut z = 4;
    z *= 2;
    println!("z *= 2: {}", z);  // 출력: z *= 2: 8
    
    // ➗ /= 연산자
    let mut w = 16;
    w /= 4;
    println!("w /= 4: {}", w);  // 출력: w /= 4: 4
    
    // 🔢 %= 나머지 연산자
    let mut m = 17;
    m %= 5;
    println!("m %= 5: {}", m);  // 출력: m %= 5: 2
}
```

**실행 결과:**
```
x += 5: 15
y -= 3: 17
z *= 2: 8
w /= 4: 4
m %= 5: 2
```

## ⚠️ 주의사항

### 1. **가변 변수(`mut`)만 사용 가능**
```rust
let x = 10;
x += 5;  // ❌ 에러! let은 불변이므로 불가능

let mut x = 10;
x += 5;  // ✅ 가능
```

### 2. **타입이 일치해야 함**
```rust
let mut x: i32 = 10;
x += 5;      // ✅ 가능 (i32 + i32)
x += 5.5;    // ❌ 에러! (i32 + f64 불가능)
```

## 🎯 실무 예제

```rust
fn main() {
    // 점수 계산
    let mut score = 0;
    score += 10;  // 정답 +10
    score += 5;   // 보너스 +5
    println!("최종 점수: {}", score);  // 출력: 최종 점수: 15
    
    // 할인 적용
    let mut price = 100;
    price -= 20;  // 20% 할인
    println!("할인 후 가격: {}", price);  // 출력: 할인 후 가격: 80
    
    // 배수 계산
    let mut power = 2;
    power *= 3;   // 2의 3제곱 (단계적)
    power *= 3;
    power *= 3;
    println!("2^3 * 3 * 3: {}", power);  // 출력: 2^3 * 3 * 3: 54
}
```

## 📌 핵심 정리

✅ `+=`, `-=`, `*=`, `/=`, `%=` 모두 지원  
✅ 코드를 더 간결하게 작성 가능  
✅ 반드시 **가변 변수(`let mut`)**에서만 사용  
✅ 성능상 차이는 없음 (가독성 개선용)
