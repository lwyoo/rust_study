## 🎭 Shadowing 예제 - 타입 변경과 가변성 제어

```rust
fn main() {
    // 1️⃣ 첫 번째 x: i32 타입
    let x = 120;
    println!("x = {}", x);  // 출력: x = 120
    
    // 2️⃣ 두 번째 x: &str 타입으로 쉐도잉 (타입 변경!)
    let x = "abc";
    println!("x = {}", x);  // 출력: x = abc
    
    // 3️⃣ 세 번째 x: bool 타입으로 쉐도잉 (이번엔 가변으로!)
    let mut x = true;
    println!("x = {}", x);  // 출력: x = true
    
    // 4️⃣ 가변 변수이므로 값만 변경 (재선언 아님)
    x = false;
    println!("x = {}", x);  // 출력: x = false
}
```

**실행 결과:**
```
x = 120
x = abc
x = true
x = false
```

---

## 📊 각 단계별 분석

| 단계 | 코드 | 타입 | 가변성 | 설명 |
|------|------|------|--------|------|
| 1️⃣ | `let x = 120;` | `i32` | ❌ 불변 | 숫자로 시작 |
| 2️⃣ | `let x = "abc";` | `&str` | ❌ 불변 | 문자열로 쉐도잉 (타입 변경!) |
| 3️⃣ | `let mut x = true;` | `bool` | ✅ 가변 | 불린으로 쉐도잉 (가변으로!) |
| 4️⃣ | `x = false;` | `bool` | ✅ 가변 | 값 변경 (재선언 아님) |

---

## 🔑 핵심 포인트

✅ **이 예제가 보여주는 것:**
- **타입 변경 가능**: `i32` → `&str` → `bool`
- **가변성 독립적**: 각 쉐도잉 시 독립적으로 가변성 결정 가능
- **이전 변수 완전 대체**: 이전의 `x`는 더 이상 접근 불가

⚠️ **주의:**
```rust
let x = 120;  // i32, 불변
let x = "abc";  // 이 순간 이전 x (120)는 메모리에서 해제됨

// let x = "abc" + 1;  // ❌ 에러! "abc"는 문자열이므로 불가능
```
